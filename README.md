Hey, welcome to Mokt!

## Supported Platforms
Mokt is written in a plugin based approach.
Some plugins and features of Mokt are not available on all platforms.

The following platforms are supported:

| Plugin         | JVM | JS | Nativ |
|----------------|-----|----|-------|
| common         | ✅   | ✅  | ✅     |
| core           | ✅   | ✅  | ✅     |
| authentication | ✅   | ❌  | ➖     |
